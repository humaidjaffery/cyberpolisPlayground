What is linear regression

The end goal of linear regression is to find a linear (straight) line that best describes a relationship between input variables (called features) and an output variable (the target). For instance, if we wanted to predict the number of people riding bikes, we might use inputs such as temperature or traffic

This process uses an existing labeled dataset (a dataset where both the inputs and the correct outputs are provided) to find a best-fit line. The best-fit line is as close as possible to all the data points in the dataset, or in other words "best-fits" the data (hence the name). This line helps us understand the relationship between the inputs (e.g., temperature) and the output (e.g., number of bikes) and allows us to make predictions for new inputs.

Best-fit line Formula

But how do we describe this line in mathematical terms? The best-fit line can described by the a simple linear function:

  • w is the slope of the line (indicating the rate of change or steepness of the line)
  • b is the y-intercept (the point where the line crosses the y-axis or how shifted up the line is)
  • x is the input variable (the temperature)
  • f(x) or y is the predicted output (number of bikes)
